<h1 style="text-align: center;">Сгенерируйте массив целых чисел, используя класс Random().</h1> 
<h2 style="text-align: left;">Используя методы классов-коллекций:</h2> 
А) создайте новый массив, переписав в него часть элементов из первого массива;<br>
Б) определите уникальные числа;<br>
В) определите количество каждого уникального числа;<br>
Г) отсортируйте массив;<br>
Д) найдите максимальное, минимальное и сумму чисел массива;<br>
Е) создайте массив, содержащий все положительные числа первого массива;<br>
Ж) удалите из первого массива все нечетные числа;<br>
З) найдите указанное число в массиве.<br>
Результат сохранить в файл. Создать итератор для коллекции.
